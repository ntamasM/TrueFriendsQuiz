## True Friends Quiz — v2.1.2

### AirConsole Compliance & Content Review

This release focuses on meeting AirConsole's developer checklist, technical requirements, and content policy for PEGI 12 compliance.

**Content Changes**

- Removed **After Dark** category (20 questions) — contained sexually explicit content
- Removed **Drunk Confessions** category (20 questions) — centered on alcohol/intoxication
- Removed **Body Count** category (20 questions) — contained sexual references
- Softened 4 answers in the **Dark Secrets** category for age-appropriateness
- All removals applied across all 7 languages (420 total questions removed)
- Game now has **420+ questions per language** across **17 categories** (9 free + 8 Pro)
- The **Spicy** vote group retains 4 categories: Deep Spicy, Dark Secrets, Never Have I Ever, Guilty

**Technical Fixes**

- Increased tap target sizes to meet 44px minimum on controller buttons (rounds +/-, settings toggles, question options, emoji reactions)
- Added loading spinners to screen and controller HTML — visible immediately on page load, auto-replaced when React mounts
- Fixed controller language independence — controller UI now uses the phone's own language instead of being overridden by the screen's language, per AirConsole requirements

**Docs & Metadata**

- Updated README, About page, and Home page with correct question/category counts
- Added version number to README and About page
- Bumped version to 2.1.2
