## True Friends Quiz — v1.2.0

### What's New

**Expanded Content**
- **200+ questions per language** (up from 40), across 7 categories: Favorites, Personality & Habits, Would You Rather, Deep & Personal, Fun & Random, Experiences & Dreams, and Deep & Spicy (Hero exclusive)
- **7 languages**: English, Greek, Spanish, French, German, Turkish, Arabic
- Questions are now loaded dynamically per language via a dedicated language loader

**New Game Settings**
- **Category toggles** — the game master can enable/disable specific question categories before starting
- **Music toggle** — background music (3 tracks) can be turned on/off from the settings panel
- **Answer time control** — configurable from 10s to 60s

**Background Music**
- 3 compressed OGG background music tracks with shuffle playback, auto-advancement, and pause/resume support

**Website & Landing Pages**
- New **How to Play** landing page (`index.html`) with step-by-step instructions and game features
- New **About** page with developer info, contribution guidelines, and supported languages
- Shared header/footer navigation and consistent styling across website pages

**Architecture & Deployment**
- Separated **app files** (website) from **game files** (AirConsole zip) into distinct directories
- Added **Dockerfile** + **nginx.conf** for containerized deployment (Coolify-ready)
- Dynamic language loading system — only the selected language is loaded at runtime

**UI Improvements**
- Redesigned settings panel with language selector, rounds control, timer control, music toggle, and category toggles
- Player count info and lobby improvements
- Leaderboard with podium display and per-player stats
